## 🚀 快速启动

### 1. 数据库
```bash
psql -U postgres -c "CREATE DATABASE cabinet_factory;"
psql -U postgres -d cabinet_factory -f scripts/init_database.sql
```

### 2. 后端
```bash
cd src/backend
npm install
cp .env.example .env
# 编辑 .env 配置数据库
npm run dev
```

### 3. 测试API
```bash
# 健康检查
curl http://localhost:3000/api/health

# 登录
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}'

# 阿尔法家导入
curl -X POST http://localhost:3000/api/alpha/validate \
  -H "Authorization: Bearer <token>" \
  -F "file=@test.xlsx"
```
