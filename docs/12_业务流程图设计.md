## 一、主业务流程（端到端）

```mermaid
flowchart TD
    subgraph 客户域
        A1[客户询价] --> A2[上门测量]
        A2 --> A3[设计方案]
        A3 --> A4[报价]
        A4 --> A5{客户确认?}
        A5 -->|否| A6[修改方案]
        A6 --> A3
        A5 -->|是| A7[签订合同]
    end

    subgraph 订单域
        A7 --> B1[支付定金]
        B1 --> B2[创建订单]
        B2 --> B3[阿尔法家拆单]
        B3 --> B4[导入板件数据]
        B4 --> B5[生成BOM]
    end

    subgraph 生产域
        B5 --> C1[材料采购]
        C1 --> C2[材料入库]
        C2 --> C3[开料切割]
        C3 --> C4[折弯加工]
        C4 --> C5[焊接打磨]
        C5 --> C6[抛光处理]
        C6 --> C7[台面生产]
        C7 --> C8[门板喷涂]
        C8 --> C9[组装]
        C9 --> C10[质检]
        C10 --> C11[包装]
        C11 --> C12[成品入库]
    end

    subgraph 交付域
        C12 --> D1[物流发货]
        D1 --> D2[上门安装]
        D2 --> D3[客户验收]
        D3 --> D4[尾款结算]
        D4 --> D5[订单完成]
    end

    subgraph 经销商域
        E1[经销商入驻] --> E2[资质审核]
        E2 --> E3[签订协议]
        E3 --> E4[经销商下单]
        E4 --> B2
        E1 --> E5[客户报备]
        E5 --> E6[设计对接]
        E6 --> A4
    end
```

## 四、采购流程

```mermaid
flowchart TD
    A[BOM分析] --> B{库存是否足够?}
    B -->|是| C[预留材料]
    B -->|否| D[生成采购计划]
    D --> E[采购审核]
    E --> F{审核通过?}
    F -->|否| G[修改计划]
    G --> D
    F -->|是| H[发送给供应商]
    H --> I[供应商报价]
    I --> J{价格合适?}
    J -->|否| K[议价/换供应商]
    K --> H
    J -->|是| L[下达采购订单]
    L --> M[材料发货]
    M --> N[到货验收]
    N --> O{验收合格?}
    O -->|否| P[退货/索赔]
    P --> H
    O -->|是| Q[入库上架]
    Q --> C
    
    C --> R[通知生产]
```

## 十、关键业务规则

### 10.1 订单状态流转

| 当前状态 | 可执行动作 | 目标状态 |
|---------|-----------|---------|
| draft | 提交 | pending |
| pending | 开始生产 | producing |
| producing | 发货 | shipped |
| shipped | 安装完成 | installed |
| installed | 验收完成 | completed |
| any | 取消 | cancelled |

### 10.2 生产阶段顺序

```
订单确认 → 设计确认 → 材料准备 → 开料切割 → 折弯 → 焊接打磨 → 抛光 → 封边 → 钻孔 → 组装 → 台面生产 → 门板喷涂 → 质检 → 包装 → 出库
```

### 10.3 经销商权限

| 功能 | 管理员 | 销售 | 设计 | 财务 |
|------|--------|------|------|------|
| 客户管理 | 全部 | 本门店 | 关联客户 | 无 |
| 订单管理 | 全部 | 查看 | 导入设计 | 查看金额 |
| 收款登记 | 是 | 否 | 否 | 是 |
| 采购管理 | 是 | 否 | 否 | 查看 |
